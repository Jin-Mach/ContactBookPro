Die Anwendung ermöglicht die Suche nach Kontakten auf zwei Arten – einfach und erweitert.

Einfache Suche  
Funktioniert nur in der markierten Spalte der Tabelle (z.. Jméno, E-mail, Telefon). Wählen Sie zuerst die Spalte in der Tabelle aus, in der Sie suchen möchten.
Geben Sie den Text in das Suchfeld ein und klicken Sie auf Vyhledat. Wenn keine Spalte ausgewählt ist, zeigt die Anwendung die Meldung "Vyberte sloupec" an und die Suche wird nicht gestartet.
Wenn Sie unseren Vorschlag aus der Autovervollständigung verwenden, wird nach einer exakten Übereinstimmung gesucht; andernfalls werden alle Einträge angezeigt, die den eingegebenen Text enthalten.

Erweiterte Suche
Durch Klicken auf Pokročilé vyhledávání wird ein Dialog geöffnet, in dem mehrere Bedingungen gleichzeitig eingegeben werden können (z.. Jméno, Město, E-mail, …).
Für jedes Feld können Sie die Option Vše, Obsahuje, Začíná oder Končí wählen.
Einzelne Bedingungen können bearbeitet oder gelöscht werden, ohne den gesamten Filter neu einstellen zu müssen.
Sie können auch eine Vorschau des aktuellen Filters anzeigen oder den Filter unter einem eigenen Namen speichern.
Nach Klick auf Hledat werden die Ergebnisse angezeigt oder eine Meldung, dass kein Eintrag den angegebenen Parametern entspricht.

Hinweis: Wiederholte Suchvorgänge durchsuchen nur Kontakte, die durch den vorherigen Filter eingeschränkt wurden. Um alle Kontakte zu durchsuchen, verwenden Sie die Option "Reset filtrů".

Gespeicherte Filter
Gespeicherte Filter können jederzeit angezeigt, verwendet oder gelöscht werden.

Filter zurücksetzen
Mit der Option Reset filtrů werden alle Bedingungen aufgehoben und alle Kontakte angezeigt.

Tastenkürzel
Vyhledávání: Ctrl + F (macOS: Cmd + F)
Pokročilé vyhledávání: Ctrl + Shift + F (macOS: Cmd + Shift + F)
Reset filtrů: Ctrl + R (macOS: Cmd + R)
Uložené filtry: Ctrl + L (macOS: Cmd + L)

Hinweis: Die Tastenkürzel funktionieren nur, wenn die Anwendung auf das Fenster "Kontakte" eingestellt ist.
