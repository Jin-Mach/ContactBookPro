Die Anwendung ermöglicht die Überprüfung von Kontakten, deren Geburtstage bevorstehen.

Beim Start überprüft die Anwendung alle Kontakte und wählt diejenigen aus, bei denen das Feld Geburtstag ausgefüllt ist und das Geburtsdatum im Bereich von ±7 Tagen zum aktuellen Datum liegt.
Am Ende wird entweder eine Benachrichtigung angezeigt, dass kein Kontakt den Kriterien entspricht, oder ein Dialog mit einer Liste der gefundenen Kontakte.
Im Dialog kann zu einem ausgewählten Kontakt gewechselt werden, wodurch der Filter auf diesen Kontakt gesetzt wird.

Die Überprüfung erfolgt im Hintergrund, sodass die Anwendung währenddessen weiterhin genutzt werden kann.

Der Benutzer wird über das Ergebnis mit einem Dialog informiert.
