Приложение позволяет выполнять поиск контактов двумя способами — простым и расширенным.

Простой поиск
Работает только в выбранном столбце таблицы (например, Имя, E-mail, Телефон). Сначала выберите в таблице столбец, в котором хотите искать.
В поле поиска введите текст и нажмите Найти. Если столбец не выбран, приложение отобразит предупреждение "Выберите столбец", и поиск не начнётся.
Если используется предложенный вариант автоподстановки, выполняется поиск точного совпадения; если нет — ищутся записи, содержащие введённый текст.

Расширенный поиск
При нажатии на Расширенный поиск открывается диалог, где можно задать несколько условий одновременно (например, Имя, Город, E-mail,...).
Для каждого поля можно выбрать опцию Все, Содержит, Начинается или Заканчивается.
Отдельные условия можно редактировать или удалять без необходимости заново настраивать весь фильтр.
Можно также отобразить предпросмотр текущего фильтра или сохранить фильтр под собственным названием.
После нажатия кнопки Искать будут показаны результаты или уведомление, что ни одна запись не соответствует заданным параметрам.

Примечание: Повторный поиск осуществляется только среди контактов, ограниченных предыдущим фильтром. Для поиска по всем контактам используйте опцию «Сброс фильтров».

Сохранённые фильтры
Сохранённые фильтры можно в любое время просмотреть, применить или удалить.

Сброс фильтров
Выбор опции Сброс фильтров отменяет все условия и отображает все контакты.

Горячие клавиши
Поиск: Ctrl + F (macOS: Cmd + F)
Расширенный поиск: Ctrl + Shift + F (macOS: Cmd + Shift + F)
Сброс фильтров: Ctrl + R (macOS: Cmd + R)
Сохранённые фильтры: Ctrl + L (macOS: Cmd + L)

Примечание: Горячие клавиши работают только если в приложении выбрано окно "Контакты".
