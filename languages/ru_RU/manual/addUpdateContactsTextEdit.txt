Форма служит для создания нового или редактирования существующего контакта.
Она содержит обязательные и необязательные поля, разделённые на несколько секций.
Перед сохранением выполняется проверка заполнения и формата данных.

Имя (текстовое поле)

Валидация и сообщения об ошибках
Если обязательные поля не заполнены, рабочий адрес неполный или формат e-mail/телефона неверен, отображается сообщение об ошибке.
Курсор автоматически устанавливается на первое ошибочное поле.
Если при сохранении нового или редактировании существующего контакта уже есть запись с таким же e-mail или номером телефона, отображается диалог со списком найденных дубликатов.
В этом диалоге можно сохранить новый контакт несмотря на дублирование, отменить сохранение или перейти к выбранному существующему контакту из списка.

После завершения отображается всплывающее окно с информацией об успешном сохранении/удалении контакта.

Получение координат
После добавления нового контакта приложение попытается автоматически получить его географические координаты (по адресу из обязательных данных).
Если координаты найдены, контакт отображается на карте.
Если координаты определить не удалось, контакт сохраняется без координат.
Приложение также содержит автоматическую проверку, которая запускается один раз через 5 минут после запуска программы.
Эта проверка пытается дополнить координаты по сохранённому адресу для всех контактов, у которых они отсутствуют.
Каждый контакт без координат проверяется таким образом максимум 20 раз.

Примечание: Для работы необходим активный доступ в интернет. Если соединение отсутствует, координаты не проверяются, но контакт будет добавлен/обновлён.

Горячие клавиши
Новый контакт: Ctrl + N (macOS: Cmd + N)
Обновить контакт: Ctrl + U (macOS: Cmd + U)

Примечание: Горячие клавиши работают только если в приложении выбрано окно "Контакты".
