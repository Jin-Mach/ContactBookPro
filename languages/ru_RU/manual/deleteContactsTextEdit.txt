Приложение позволяет удалять как отдельные контакты, так и все контакты сразу.

Удаление одного контакта
Выберите нужный контакт в таблице и нажмите кнопку удаления.
Появится окно подтверждения с вопросом "Действительно удалить?" и возможностью подтвердить или отменить удаление.
После подтверждения выбранный контакт будет удалён, и появится всплывающее окно с информацией об успешном удалении контакта.

Удаление всех контактов
Не нужно ничего выделять – достаточно нажать кнопку удаления всей базы данных.
Откроется окно подтверждения с текстовым полем, в которое необходимо ввести слово "smazat" для подтверждения действия.
В этом окне также можно отметить опцию для удаления сохранённых фильтров.
Если опция включена, будут удалены все сохранённые фильтры.
После подтверждения приложение попытается удалить все контакты из базы данных, стереть сохранённые фильтры (если опция включена) и удалить файл базы данных (contacts_db.sqlite).
Если удалить файл базы данных не удастся, будет очищено только его содержимое, а сам файл останется.
Затем приложение автоматически перезапустится и создаст новую пустую базу данных.

!!! Не удаляйте файл базы данных вручную, это приведёт к безвозвратной потере данных !!!

Горячие клавиши
Удалить выбранный контакт: Ctrl + D (macOS: Cmd + D)
Удалить всю базу данных: Ctrl + Shift + D (macOS: Cmd + Shift + D)

Примечание: Горячие клавиши работают только если в приложении выбрано окно "Контакты".
