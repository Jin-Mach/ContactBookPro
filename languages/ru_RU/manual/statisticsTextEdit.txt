Приложение предлагает обзор статистики контактов, разделённый на несколько вкладок, которые визуализируют различные аспекты базы данных с помощью графиков.

Основная статистика
Круговая диаграмма отображает распределение контактов по полу.
Столбчатая диаграмма показывает распределение контактов по отношениям.
Столбчатая диаграмма отображает самые распространённые города контактов.

Рабочая статистика
Две круговые диаграммы показывают процентное заполнение необязательных полей E-mail и Телефон.
Столбчатая диаграмма показывает количество контактов по городам на основе рабочих данных.

Социальная статистика
Столбчатая диаграмма, где каждый столбец разделён на две части — количество заполненных и незаполненных записей для отдельных социальных сетей.

Системная статистика
Верхняя горизонтальная столбчатая диаграмма показывает процентное заполнение всей базы данных — соотношение всех возможных полей к количеству заполненных.
Нижняя горизонтальная столбчатая диаграмма отображает то же самое для отдельных таблиц базы данных.

После добавления, изменения или удаления контакта пользователь уведомляется о изменении статистики кратким сообщением в нижней части приложения.
