Приложение позволяет экспортировать выбранный контакт в формате vCard.

Экспорт в формат vCard (.vcf) позволяет сохранить контакты в стандартизированном формате, который поддерживают мобильные телефоны, почтовые клиенты (например, Microsoft Outlook, Thunderbird) и другие приложения для управления контактами.
Каждый экспортируемый контакт содержит основные данные — полное имя, e-mail, номер телефона и личный адрес.
После выбора экспорта в vCard отображается диалог для указания расположения и имени файла.
По умолчанию диалог открывается в папке Документы (одно и то же расположение для Windows и macOS), но пользователь может выбрать любую другую папку.

Экспорт выполняется мгновенно.

После завершения отображается диалог с информацией об успешном сохранении.
